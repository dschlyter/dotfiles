C-b ? - list all keys
C-b (repeated) - send C-b to inner program (ie. vim or nested tmux from ssh)

C-b space - next layout
C-b M-1,2,3,4,5 - predefined layouts
C-b M-Left,Right,Up,Down - resize
C-b {,},C-o,A-o - move or rotate pane
C-b z - toggle zoom

C-b c , - create window and name window
C-b 0-9 n p l - navigate windows

C-b :new - create session
C-b s L ( ) - navigate to other sessions
C-b $ - name the session

C-b [ ] - enter copy-mode, paste
C-b C-c - enter copy-mode
C-b P - paste

C-b x & - kill pane/window
